# Getting Started with CatchBack

Welcome to CatchBack! This guide will help you get up and running with our returns management platform quickly and efficiently.

## Quick Setup

### 1. Create Your Account

<Steps>
  <Step title="Sign Up">
    Visit [app.catchback.com](https://app.catchback.com) and create your merchant account
  </Step>
  <Step title="Verify Email">
    Check your email and click the verification link
  </Step>
  <Step title="Complete Profile">
    Add your business information and return policies
  </Step>
  <Step title="Choose Plan">
    Select the plan that best fits your business needs
  </Step>
</Steps>

### 2. Connect Your Store

<Tabs>
  <Tab title="Shopify">
    1. Go to the Shopify App Store
    2. Search for "CatchBack Returns"
    3. Click "Add app" and authorize permissions
    4. Configure your return settings
  </Tab>
  <Tab title="WooCommerce">
    1. Download the CatchBack plugin
    2. Install and activate in your WordPress admin
    3. Navigate to WooCommerce â†’ CatchBack Settings
    4. Enter your API credentials
  </Tab>
  <Tab title="API Integration">
    1. Generate API credentials in your CatchBack dashboard
    2. Review our [Integration Guide](/technology/integration)
    3. Implement the API endpoints
    4. Test with our sandbox environment
  </Tab>
</Tabs>

### 3. Configure Your Settings

#### Return Policies
Set up your return rules and timeframes:

```javascript
// Example return policy configuration
const returnPolicy = {
  defaultWindow: 30, // days
  categories: {
    'electronics': { window: 14, restockingFee: 0.15 },
    'clothing': { window: 45, restockingFee: 0 },
    'books': { window: 7, restockingFee: 0 }
  },
  conditions: ['new', 'opened', 'damaged'],
  autoApproval: {
    enabled: true,
    maxValue: 100,
    conditions: ['new', 'opened']
  }
};
```

#### Notification Settings
- Customer email templates
- SMS notifications (optional)
- Merchant alert preferences
- Return status updates

## Best Practices

### Policy Setup
- **Clear Guidelines**: Set specific, easy-to-understand return conditions

## Troubleshooting

### Common Issues

<Accordion>
  <AccordionItem title="API Authentication Errors">
    - Verify API credentials are correct
    - Check if sandbox/production mode matches
    - Ensure proper request headers are included
    - Validate token expiration
  </AccordionItem>
  
  <AccordionItem title="Return Eligibility Issues">
    - Review return policy configuration
    - Check product category settings
    - Verify time window calculations
    - Confirm order status requirements
  </AccordionItem>
  
  <AccordionItem title="Email Notifications Not Sending">
    - Verify email templates are configured
    - Check spam/junk folders
    - Confirm notification preferences
    - Review email service provider settings
  </AccordionItem>
</Accordion>
